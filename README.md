# Heartbleed_java

Exploit code for OpenSSL Heartbleed bug. Written in Java.

## usage

### build
   
     % mvn install

### execute

     % java -jar target/HeartBleedChecker.jar <host>

### with specific port (default: 443)

     % java -jar target/HeartBleedChecker.jar <host> <port>
